# athenaq

## install
```
$ git clone https://github.com/ritarock/athenaq.git
$ cd athenaq
$ go install
```

## Usage
```
NAME:
   athenaq - Run athena query

USAGE:
   athenaq [global options] command [command options] [arguments...]

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --profile value  set aws profile
   --bucket value   set aws bucket
   --key value      set aws bucket key
   --query value    set athena query
   --file value     set file athena query
   --help, -h       show help (default: false)
```
